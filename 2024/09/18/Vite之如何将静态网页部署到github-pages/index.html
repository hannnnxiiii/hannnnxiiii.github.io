如何把多个项目塞进 GitHub Pages？

注 1：本文假设你已经有了属于自己的 username.github.io 仓库，如果你不知道这个是什么，可以先去搜索引擎搜索了解一下。

注 2：本文假设你用的构建工具是 Vite，其他的实现方式应该相似。

把多个项目塞进 GitHub Pages，查遍了互联网的教程都没做成功，特此留档

  1. 生成静态文件

    npm run build

  2. 修改 vite.config.js

  3. 安装 gh-pages

    1. 安装

      npm install gh-pages --save-dev

    2. 修改 package.json

  4. 构建并部署

    在此之前请先确定你已经和远程仓库建立好连接，接着用以下代码构建并部署

    npm run build
    npm run deploy

  5. 配置 GitHub Pages

    • 打开你的 GitHub 仓库页面，点击 Settings。

    • 在左侧的导航栏中找到 Pages。

    • 在 “Source” 下拉菜单中选择 gh-pages 分支，然后点击保存。

    GitHub 会自动将 gh-pages 分支的内容发布为静态网站。

此时你的网站就可以访问了，地址应该为：https://your-username.github.io/your-repository/
